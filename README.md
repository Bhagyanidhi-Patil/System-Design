# 🚀 System Design Repository (HLD + LLD in C++)

This repository is a structured collection of **High-Level Design (HLD)** concepts and **Low-Level Design (LLD)** implementations using **C++**.

It is designed to help you:

* Understand **scalable system architecture**
* Master **object-oriented design principles**
* Learn and implement **design patterns**
* Prepare for **system design interviews**

---

## 📚 What’s Inside

### 🔹 High-Level Design (HLD)

* Scalability concepts
* System architecture fundamentals
* Distributed system basics

### 🔹 Low-Level Design (LLD)

* Object-oriented design problems
* Clean code practices
* Real-world class design

### 🔹 Design Patterns (C++)

* Creational patterns
* Structural patterns
* Behavioral patterns
